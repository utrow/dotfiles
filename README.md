# dotfiles

## Import this dotfiles

1. Clone repository

1. Run script
    ```
    bash init.sh
    ```


## Manual Installations

### Applications

- Chrome
- Homebrew
- Karabiner
- iTerm
    - import profile and color scheme from ./iterm2
- VSCode
- Docker

### Font

- Source Han Mono
